# Roadmap

## Now

- Keep the source-install workflow reliable.
- Keep the tiny demo fast and reproducible.
- Keep CI aligned with local release gates.

## Next

- Improve language-specific test-file detection.
- Add more task-curation examples.
- Document optional container execution patterns.

## Later

- Add richer failure categorization.
- Add adapter examples for common coding-agent CLIs.
- Consider export formats for external benchmark tooling.

## Non-goals

- Cloud leaderboard.
- Replacing SWE-bench.
- Automatic execution of untrusted remote repositories.
- Fake model comparisons.
- Publishing private code.
