# Release Notes

## 0.1.0

PatchGym 0.1.0 is a source-install alpha for local coding-agent benchmark generation. It is not published to PyPI.

Highlights:

- mine real Git history into benchmark tasks,
- verify base-fails/oracle-passes validity,
- run shell-command agents,
- generate JSON, Markdown, and HTML reports,
- run a no-key tiny repository demo.
