# Release Notes

## 0.1.0 - Draft

PatchGym 0.1.0 is planned as a source-install alpha. Do not create a GitHub
release or publish to PyPI until CI has passed on the release commit and the
maintainer explicitly approves the release.

Highlights:

- mine real Git history into local coding-agent tasks,
- verify base-fails/oracle-passes task validity,
- run any explicit shell-command agent,
- generate JSON, Markdown, and HTML reports,
- run a no-key tiny demo.
